# BASIC-HEALTH-MANAGEMENT-SYSTEM
🏥 Basic Health Management System

📱 Overview The Basic Health Management System is an Android application developed as part of a Mobile Application Development Lab (MAD Lab) project. It helps users manage essential daily health activities like water intake, medicine tracking, and fitness planning in a simple and user-friendly way.

✨ Features

💧 Water Tracker •Set daily water intake goal (in glasses) •Set reminder interval (for display purposes) •Track water consumption using a progress bar •Data stored using SharedPreferences

💊 Medicine Tracker •Add medicine name and dosage •Enter time (Morning / Afternoon / Evening) •AM/PM toggle support •View saved medicines in a list •Delete medicines easily

⚠️ Note: This module is UI-based (no real notifications)

🏋️ Fitness Planner •Enter weight, height, and age •Automatic BMI calculation •Get diet plan (Veg / Non-Veg) •Get workout plan (Cut / Bulk) •Simple recommendation system

🚨 SOS Feature One-click emergency dial (112) 🛠️ Tech Stack •Language: Kotlin •UI Design: XML •IDE: Android Studio •Storage: SharedPreferences
